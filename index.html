MoneyWiz

Functional Prototype Usability Test


The participants will perform a set of predefined tasks using the MoneyWiz mobile application prototype.

During the experiment, data will be collected regarding their demographic profile and their comments and difficulties on performing the tasks and using the application, overall.


Tasks to perform:
  • Task 1 - Add a new budget for the account 'Poupanças CGD' and a category of your choice
  • Task 2 - Change the name and description of account 'Cartão CGD' from CGD to Santander
  • Task 3 - Add a new income category to the system
  • Task 4 - Check how much was earned in February 2020 in the category 'Salary'
  • Task 5 - Add a new 'Restaurants' expense transaction for the 28th of January at 20:25
  • Task 6 - Check the balance for the week of April 20-26
  • Task 7 - Remove the expenses category 'Taxes' from the system
